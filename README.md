# TA_PROGDAS_Radhitya-Kurnia-Asmara_21120125130084_Forensic-Analyzer-Dokumen-Office
aplikasi GUI Python untuk menganalisis keamanan file Office (.docx, .docm, .dotx, .dotm), mendeteksi macro malware, menemukan file dan payload tersembunyi, mengekstraksi embedded objects, serta menilai tingkat risiko melalui analisis struktur dokumen, metadata, entropy, dan teknik polyglot.
